Table tags

  1. <a href="я ссылка"></a>
    Определяет гиперссылку
  2. <!--...-->
    Определяет комментарий
  3. <!DOCTYPE>
    Определяет тип документа
  4. <abbr>
    Определяет абривеатуру
  5. <acronym>
    Не поддерживается в HTML5. Вместо этого используйте "abbr". Определяет аббревиатуру
  6. <address>
    Определяет контактную информацию для автора/владельца документа
  7. <area>
    Определяет область внутри карты изображения
